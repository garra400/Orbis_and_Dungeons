# Orbis and Dungeons - Race Selection Mod

## Overview
Orbis and Dungeons introduces a permanent race selection system to Hytale. Upon entering the world for the first time, players must choose between three unique races, each offering distinct gameplay advantages through modified stats.

## Features
- **One-time race selection** - Choose your race when first entering the world
- **Permanent choice** - Once selected, your race cannot be changed
- **Balanced gameplay** - Each race excels in different playstyles
- **Custom UI** - Beautiful race selection interface with detailed descriptions
- **No portal interference** - Selection screen only appears on first world entry, not when traveling between dimensions

## Technical Details

**Base Game Stats:**
- Health: 100
- Stamina: 10

**Race Modifications:**
- All bonuses apply to MAX stats (Health/Stamina caps)
- Bonuses are additive
- Stats are applied once upon race confirmation
- No mana modifications (only Health and Stamina affected)

## Installation
1. Download the mod JAR file
2. Place it in your Hytale mods folder: `UserData/Saves/mods/mods/`
3. Launch Hytale
4. Enter your world and choose your race!

## Compatibility
- Works with all armor types
- Compatible with vanilla stat systems
- No conflicts with dimension travel
- Session-persistent (choice remains until server restart)
